STEP 1: OPEN POWERSHELL AS ADMINISTRATOR
-----------------------------------------
1. Press Windows Key
2. Type "PowerShell"
3. Right-click "Windows PowerShell"
4. Click "Run as Administrator"


STEP 2: INSTALL GIT (If not installed)
---------------------------------------
1. Download Git from: https://git-scm.com/download/win
2. Run installer, click Next â†’ Next â†’ Install
3. Restart PowerShell


STEP 3: CLONE THE REPOSITORY
-----------------------------
In PowerShell, run these commands:

cd C:\
git clone https://github.com/JaePyJs/CLMS.git
cd CLMS


STEP 4: CHECK WHAT YOU GOT
---------------------------
Run this to see all files:

dir

You should see:
- Backend/ folder
- Frontend/ folder
- docker-compose.yml
- .env.production
- csv/ folder (with your book & student files)
- INSTALL_AT_LIBRARY.txt
- LIBRARY_DEPLOYMENT.md


STEP 5: SWITCH TO CORRECT BRANCH
---------------------------------
Run:

git checkout 002-frontend-stability-auth-testing


STEP 6: VERIFY CSV FILES ARE THERE
-----------------------------------
Run:

dir csv

You should see:
âœ… SHJCS Bibliography - BOOK COLLECTIONS.csv
âœ… SHJCS SCANLOGS - SHJCS USERS.csv


STEP 7: SETUP ENVIRONMENT FILE
-------------------------------
Run:

copy .env.production .env


STEP 8: INSTALL DOCKER DESKTOP
-------------------------------
NOW you need Docker to run the system:

1. Download from: https://www.docker.com/products/docker-desktop/
2. Run installer
3. Restart computer
4. Start Docker Desktop


STEP 9: START THE SYSTEM
-------------------------
Back in PowerShell (in C:\CLMS folder):

docker-compose up -d

Wait 2 minutes for everything to start.


STEP 10: IMPORT DATA
--------------------
Run these commands:

docker-compose exec backend npx prisma migrate deploy
docker-compose exec backend node src/scripts/importCsv.js


STEP 11: CREATE ADMIN ACCOUNT
------------------------------
Run:

docker-compose exec backend node create-librarian.js

Follow the prompts to create your admin account.


STEP 12: GET YOUR IP ADDRESS
-----------------------------
Run:

ipconfig

Look for "IPv4 Address" - Example: 192.168.0.126


STEP 13: ACCESS THE SYSTEM
---------------------------
Open Chrome browser and go to:

http://YOUR-IP:8080

Example: http://192.168.0.126:8080

Login with the account you created in Step 11.
