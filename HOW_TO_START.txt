# HOW TO START CLMS

## ONE-CLICK STARTUP:

Double-click: `START_CLMS.bat`

That's it!

---

## What You'll See:
- 2 terminal windows will open (DO NOT CLOSE)
- Browser opens automatically to http://localhost:3000
- Login page appears

## To Stop:
- Close both terminal windows
- Or press Ctrl+C in each window

## Database Backup:
- Copy file: `Backend/prisma/dev.db`
- Paste to safe location

---

## Common Errors:
- "EADDRINUSE" or "Port already in use":
  Means the server is ALREADY running!
  SOLUTION: Close ALL black terminal windows and try again.

Questions? The app tells you everything when it starts!
