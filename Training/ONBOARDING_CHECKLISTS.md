## New Staff Onboarding Overview

### Onboarding Philosophy
The CLMS onboarding program is designed to:
- Ensure smooth transition into library operations
- Build confidence and competence with system usage
- Establish understanding of library policies and procedures
- Create positive first impressions and engagement
- Provide ongoing support for continuous learning

### Training Objectives
By the end of the onboarding period, new staff will be able to:
- Navigate CLMS interface efficiently
- Perform daily library operations independently
- Troubleshoot common system issues
- Provide excellent customer service
- Understand library policies and procedures

### Training Timeline
- **Pre-Arrival**: 1 week before start date
- **Day 1**: Orientation and basic system access
- **Week 1**: Core functions and daily operations
- **Week 2**: Advanced features and role-specific tasks
- **Week 3**: Independent practice and assessment
- **Week 4**: Full integration and support transition

## First Week Training Plan

### Day 1: Orientation and Basic Navigation
**Focus**: Welcome, system access, basic navigation
**Trainer**: Library Director/Head Librarian
**Topics Covered**:
- Library mission and organizational structure
- CLMS login and password setup
- Basic interface navigation
- Keyboard shortcuts and help system
- Building tour and facilities orientation

### Day 2: Student Management
**Focus**: Student records, search, basic data entry
**Trainer**: Senior Librarian
**Topics Covered**:
- Student database structure and fields
- Adding and editing student records
- Search and filter techniques
- Student status management
- Bulk operations and data export

**Practice Exercises**:
- Add 5 sample student records
- Search for students using various criteria
- Practice editing student information
- Export student data to Excel

### Day 3: Book Catalog and Circulation
**Focus**: Book search, checkout, return procedures
**Trainer**: Circulation Supervisor
**Topics Covered**:
- Book catalog structure and organization
- Book search and filtering
- Standard checkout procedures
- Return processing and condition assessment
- Due date calculation and fine structure

**Practice Exercises**:
- Search for books using different methods
- Process 10 sample checkouts
- Process 5 sample returns
- Handle various return conditions

### Day 4: Equipment Management
**Focus**: Equipment checkout, tracking, basic maintenance
**Trainer**: Equipment Coordinator
**Topics Covered**:
- Equipment categories and availability
- Equipment checkout procedures
- Condition assessment and reporting
- Basic maintenance and troubleshooting
- QR code usage and mobile integration

**Practice Exercises**:
- Check out 3 different equipment types
- Process equipment returns with various conditions
- Generate equipment usage report
- Practice equipment maintenance procedures

### Day 5: Activity Tracking and Self-Service
**Focus**: Student activity tracking, self-service operations
**Trainer**: Library Services Manager
**Topics Covered**:
- Activity types and tracking procedures
- Manual vs. automated tracking
- Self-service configuration and monitoring
- Real-time activity monitoring
- Activity report generation

**Practice Exercises**:
- Track 10 sample student activities
- Configure self-service station
- Monitor real-time activity dashboard
- Generate activity summary report

### End of Week Review

Week 1 Assessment:
