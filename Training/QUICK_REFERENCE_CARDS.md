## Student Management Quick Guide

### Adding New Students
```
1. Navigate to Student Management (Alt+2)
2. Click "Add Student"
3. Enter required information:
   • Student ID (YYYY + 4 digits)
   • First Name
   • Last Name
   • Grade Level
   • Section
4. Add optional information (email, phone, etc.)
5. Click "Save Student"
6. Generate barcode if desired
```

### Student Search Tips
```
• Search by student ID, name, grade, or section
• Results update as you type
• Use Advanced Search for complex queries
• Filter by status: Active/Inactive/All
• Filter by grade category
• Use checkboxes for bulk operations
```

### Common Student Tasks
```
• Edit Student: Click Edit button next to record
• Change Status: Click "Change Status" button
• Generate Barcodes: Select students → "Generate Barcodes"
• Export Data: Select students → "Export"
• Deactivate: Select students → "Bulk Deactivate"
• View History: Click on student name
```

### Student ID Format
```
Format: YYYY + 4-digit number
Example: 2023001 (First student in 2023)

Grade Categories (Auto-calculated):
• PRIMARY: Grades K-2
• GRADE_SCHOOL: Grades 3-6
• JUNIOR_HIGH: Grades 7-9
• SENIOR_HIGH: Grades 10-12
```

## Book Checkout Process

### Standard Checkout
```
1. Navigate to Book Checkout (Alt+4)
2. Enter/Scan Student ID
3. Verify student account status
4. Enter/Scan Book Accession Number
5. Set due date (auto-calculated)
6. Add special notes if needed
7. Click "Checkout Book"
8. Print receipt if requested
```

### Book Return Process
```
1. Navigate to Book Checkout tab
2. Enter/Scan Accession Number
3. Select return condition:
   • Good (no new damage)
   • Damaged (new damage noted)
   • Lost (book not returned)
4. Add condition notes
5. Click "Return Book"
6. Process fines if applicable
```

### Due Date Rules
```
• Standard Books: 14 days
• Reference Materials: 3 days
• New Releases: 7 days
• Holiday Extensions: Automatic
• Staff Loans: 30 days
• Renewals: Allowed if no reservations
```

### Fine Structure
```
• Standard Overdue: $0.10 per day
• Reference Materials: $0.25 per day
• New Releases: $0.50 per day
• Maximum Fine: Equal to book value
• Grace Period: None (strict enforcement)
```

### Common Issues
```
• Can't find student: Check spelling, try different search
• Book not found: Verify accession number, check status
• Account blocked: Check for overdues or fines
• Scanner not working: Check connection, restart scanner
• System slow: Clear cache, restart browser
```

## Equipment Management

### Equipment Checkout
```
1. Navigate to Equipment Dashboard (Alt+5)
2. Enter/Scan User ID
3. Verify user eligibility
4. Search for available equipment
5. Select equipment to check out
6. Specify purpose of use
7. Review checkout details
8. Click "Checkout Equipment"
```

### Equipment Return
```
1. Enter/Scan Equipment ID
2. Verify checkout information
3. Assess equipment condition:
   • Good: No new damage
   • Minor Issues: Small problems noted
   • Damaged: New damage requires attention
   • Needs Repair: Equipment not fully functional
4. Add condition notes
5. Click "Return Equipment"
6. Schedule maintenance if needed
```

### Equipment Categories
```
• Computer Equipment: Desktops, laptops, tablets
• Audio-Visual: Projectors, speakers, cameras
• Office Equipment: Printers, scanners, copiers
• Furniture: Chairs, tables, carrels
• Specialized: 3D printers, VR equipment
```

### Time Limits
```
• Computers: 2 hours maximum
• Laptops/Tablets: 2 hours, library use only
• Audio-Visual: Varies by equipment
• Group Study Rooms: 2 hours maximum
• Special Equipment: As specified
```

### Maintenance Procedures
```
• Report damage immediately
• Set status to "Under Maintenance"
• Create maintenance request
• Track repair progress
• Document all maintenance
• Schedule preventive maintenance
```

## Activity Tracking

```
1. Navigate to Scan Workspace (Alt+6)
2. Enter Student ID or scan barcode
3. Select activity type:
   • Study (default)
   • Computer Use
   • Research
   • Reading
   • Group Work
   • Homework
4. Click "Start Activity"
```

### Ending Activity
```
1. Find student in active sessions
2. Click "End Activity" button
3. Confirm activity completion
4. System calculates duration
5. Activity logged in student history
```

### Self-Service Tracking
```
Student Process:
• Scan ID card for check-in
• System automatically starts activity
• Default activity type applied
• Welcome message displayed

Check-out Process:
• Scan ID card again
• System detects active session
• Activity automatically ended
• Session duration calculated
```

### Activity Types
```
• Study: Individual study and homework
• Computer Use: Computer workstation usage
• Research: Information gathering
• Reading: Book reading
• Group Work: Collaborative activities
• Homework: Assignment completion
• Tutoring: Academic support sessions
```

### Time Limits
```
• Standard Sessions: 2 hours maximum
• Extended Sessions: Staff approval required
• Cooldown Period: 30 minutes between scans
• Automatic Check-out: At closing time
• Self-Service Hours: Configurable
```

## Troubleshooting Guide

### Login Issues
```
Problem: Can't log in
Solutions:
• Check username and password spelling
• Verify account is active
• Clear browser cache and cookies
• Try different browser
• Contact administrator if locked out

Problem: Account locked
Solutions:
• Wait 15 minutes for automatic unlock
• Contact administrator for immediate unlock
• Verify correct login credentials
```

### Scanner Issues
```
Problem: Scanner not reading barcodes
Solutions:
• Check USB connection
• Verify scanner is in keyboard wedge mode
• Test with different barcodes
• Check barcode quality and contrast
• Restart scanner and computer

Problem: Scanner scans but no data appears
Solutions:
• Check cursor is in correct field
• Verify scanner is in keyboard wedge mode
• Test with different applications
• Check for conflicting input devices
```

### Performance Issues
```
Problem: System running slowly
Solutions:
• Check internet connection speed
• Clear browser cache and cookies
• Close unnecessary browser tabs
• Restart browser
• Check system status page

Problem: Pages not loading completely
Solutions:
• Refresh page (F5 or Ctrl+R)
• Check browser compatibility
• Disable browser extensions
• Try different browser
• Verify internet connection
```

### Data Issues
```
Problem: Can't find student records
Solutions:
• Check spelling of search terms
• Try different search variations
• Verify student is in correct grade level
• Check for inactive status
• Search by student ID number

Problem: Duplicate records
Solutions:
• Identify correct record
• Report duplicate to administrator
• Do not delete duplicates yourself
• Document duplicate record issues
```

## Barcode Scanner Tips

### Setup and Connection
```
• Connect USB scanner to computer
• Ensure scanner is in keyboard wedge mode
• Test scanner in text editor first
• Check indicator light (should be solid green)
• Configure auto-enter delay if needed
```

### Scanning Techniques
```
• Hold scanner 3-6 inches from barcode
• Position scanner at 45-60 degree angle
• Ensure good lighting on barcode
• Move scanner smoothly across barcode
• Listen for confirmation beep
```

### Maintenance
```
• Clean scanner window regularly
• Use soft, dry cloth for cleaning
• Avoid liquid cleaners
• Check cable connections
• Protect from drops and damage
```

### Troubleshooting
```
No response:
• Check USB connection
• Verify power (indicator light)
• Test different USB port
• Restart computer

Reads but wrong data:
• Check scanner configuration
• Verify barcode format
• Update scanner drivers
• Reconfigure settings

Beeps but no data:
• Check cursor in correct field
• Verify keyboard wedge mode
• Test with different applications
```

## Emergency Procedures

### System Outage
```
Immediate Actions:
• Switch to manual procedures
• Use paper circulation logs
• Document all transactions
• Notify system administrator
• Post outage notice for patrons

Manual Checkout:
• Record student ID and book accession number
• Note checkout time and due date
• Collect contact information if possible
• Explain delayed processing to patrons

Recovery:
• Enter all manual transactions when system restored
• Notify patrons of any delays
• Review and verify entered data
• Update any affected records
```

### Security Incidents
```
Suspicious Activity:
• Do not confront suspicious individuals
• Contact security or school administration
• Document what you observed
• Preserve any evidence
• Follow school security procedures

System Security Issues:
• Stop using system immediately
• Contact IT administrator
• Document error messages
• Preserve system logs
• Follow incident response procedures
```

### Medical Emergency
```
• Call emergency services (911) if needed
• Notify school nurse or administration
• Clear area if necessary
• Direct emergency responders
• Document incident after resolved
• Follow school medical emergency procedures
```

### Fire or Evacuation
```
• Activate fire alarm if needed
• Evacuate building immediately
• Assist patrons as needed
• Take attendance sheet if possible
• Meet at designated assembly point
• Follow school evacuation procedures
• Account for all patrons and staff
```

### Lost or Damaged Items
```
Lost Books:
• Mark as lost in system
• Calculate replacement cost
• Notify patron of charges
• Update patron account
• Document circumstances

Damaged Equipment:
• Remove from service immediately
• Create maintenance request
• Document damage
• Notify appropriate staff
• Follow reporting procedures
```

## Printing Instructions

These quick reference cards are designed to be printed on standard 8.5" x 11" paper and can be:

1. **Printed as a complete reference guide** (double-sided, bound)
2. **Printed as individual cards** (one section per card, laminated)
3. **Made into a flip chart** for easy reference at circulation desk
4. **Saved as PDF** for digital access on computers and tablets

### Recommended Printing Settings
- **Paper**: Standard 8.5" x 11" or A4
- **Orientation**: Portrait
- **Color**: Black and white or color as preferred
- **Lamination**: Recommended for heavy use
- **Binding**: 3-hole punch for binder storage

These quick reference cards provide essential information for daily CLMS operations and should be kept readily available at all library service points.
