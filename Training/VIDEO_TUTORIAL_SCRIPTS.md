## Production Guidelines

### General Production Standards

#### Video Specifications
- **Duration**: 3-8 minutes per tutorial
- **Resolution**: 1920x1080 (Full HD)
- **Format**: MP4 with H.264 encoding
- **Frame Rate**: 30 frames per second
- **Audio Quality**: Clear voice-over with background music (optional)

#### Recording Setup
1. **Screen Recording**:
   - Use system audio recording
   - Show mouse cursor clearly
   - Highlight important elements
   - Use smooth transitions between steps

2. **Voice Guidelines**:
   - Speak clearly and at moderate pace
   - Use professional but friendly tone
   - Avoid technical jargon when possible
   - Include pause breaks for viewer comprehension

3. **Visual Elements**:
   - Zoom in on important details
   - Use callout boxes for emphasis
   - Include progress indicators
   - Show keyboard shortcuts when used

#### Post-Production
1. **Editing Standards**:
   - Add chapter markers for navigation
   - Include closed captions
   - Add progress bar
   - Include end screens with resources

2. **Branding**:
   - Include library logo and branding
   - Use consistent color scheme
   - Add contact information
   - Include copyright notices
